## How to run ##
	# add your apikey
在src/apikey.js 中修改成你的 apikey <br/>
(可以去[APIStore](http://apistore.baidu.com/ "APIStore")那申请，很简单快速。）

    # install dependencies 
    npm install 
    
    # run server
    node app
