## How to run ##
	# add your apikey
在components/weather.vue 和 technology_news.vue 中数据中的 apikey
修改成你的apikey <br/>
(可以去[APIStore](http://apistore.baidu.com/ "APIStore")那申请，很简单快速。）

    # install dependencies 
    npm install
    
    # run server
    npm run build 
    
打开builds文件夹 的app.html
